# 漂泊ノ海 手紙投函フォーム サンプル

## 確認方法

このフォルダでローカルHTTPサーバーを起動してください。

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## 現在入っているもの

- 元サイトと共通する暗い青・半透明カード・明朝体中心の世界観
- 差出人、標題、本文の入力
- 文字数表示と必須チェック
- `localStorage` を使った端末内の下書き保存
- 投函前の確認画面
- 海へ手紙を流すモック演出
- スマートフォン向けレスポンシブ表示
- Cloudflare Turnstileの設置予定領域
